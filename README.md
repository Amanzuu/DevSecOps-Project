# 🚀 DevSecOps Project - Task Manager API

## 📌 Features
- REST API (CRUD operations)
- Built with Node.js & Express
- Tested using Postman

## 🛠️ Tech Stack
- Node.js
- Express.js

## 🔗 API Endpoints
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

npm install
node index.js
